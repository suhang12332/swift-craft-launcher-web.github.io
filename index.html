Swift Craft Launcher

Swift Craft Launcher is a native macOS Minecraft launcher built with SwiftUI, delivering a smooth and efficient gaming experience. Designed for modern macOS systems, it integrates full mod loader support, Microsoft account authentication, and intuitive game management features.

Requires macOS 14.0 or later
